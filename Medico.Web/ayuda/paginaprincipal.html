Página Principal

Contiene las funciones comunes de la página principal de la aplicación y la navegación en las opciones que el Administrador del Sistema le ha proporcionado permiso para utilizar:

La página principal tiene las siguientes partes:

▪El encabezado.

Contiene el control hamburguer Menu para Ocultar o mostrar el menú, el logo con el nombre de la aplicación y el enlace para mostrar el detalle de su cuenta de usuario

▪El Arbol del Menú. Se encuentra en la parte izquierda de la página y se oculta o muestra dando clic en el control  

▪El área o panel donde se muestran las consultas o mantenimientos

▪El Pie con el nombre y versión del sistema

 

Tips

Puede Ocultar el árbol del menú dando clic en el control Slide Out Menu, cuando el menú esta visible. Clic en el mismo control para mostrar el menú cuando esta oculto. Esta funcionalidad es útil para ampliar el área de despliegue de los datos.
