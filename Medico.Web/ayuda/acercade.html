Acerca del Sistema

Este documento tiene como propósito describir la funcionalidad del Sistema de Registro Electrónico de Pacientes. Es una herramienta para la administración electrónica de los expedientes clínicos y está orientada a medicos, clínicas y pequeños hospitales. Las funcionalidades generales de la aplicación son las siguientes:

•Multi Empresa y Multi Consultorio.

•Multi Medico.

•Versiones con Interfaz Web y Windows.

•La versión web puede usarse en dispositivos móviles como tablets y smartphones.

•Componente de seguridad.

•Mantenimientos y configuración de catálogos generales del sistema.

•Agenda de las citas por médico.

•Expediente clínico de los pacientes.

•Registro de las consultas.

•Las versiones premium de la aplicación poseen además las siguientes funcionalidades.

oDiseñador de Reportes.

oEditor de vistas.

oConsultas Multidimencionales (BI)

 

Otras características de la aplicación son las siguientes:

•Catálogo de países ISO 3166-1 Alpha 3.

•Departamentos y Municipios preconfigurados para El Salvador.

•Clasificación Internacional Uniforme de Ocupaciones (CIUO) para las profesiones y oficios.

•Catálogo CIE-10 para la codificación de los padecimientos.

•Catálogo CVX para las vacunas.

•Listado de Medicamentos LIME (con códigos ATQ, versión del año 2016.

 

Todos los catálogos mencionados acá se crean por defecto al momento de instalar la aplicación. Si no los desea, el técnico responsable de la implementación podrá crear durante la instalación una base de datos vacía o cargar solamente aquellos que son útiles para sus necesidades particulares. En todo caso requerirá tiempo extra de configuración y podría haber cargos adicionales.
