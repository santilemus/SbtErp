Requisitos

El Registro Electrónico de Pacientes esta disponible en dos ediciones cliente servidor:

•Windows. Todos los equipos que utilizan el sistema, deben tener instalado un sistema operativo Windows, versión 7 o posterior.

•Web. Tiene la ventaja que puede usarse en dispositivos móviles como tablets y smartphone. Además es independiente del SO, por lo tanto puede usarse en equipos que ejecuten cualquier sistema operativo, siempre y cuando tengan un navegador Web moderno. La desventaja es que necesita un servidor con recursos apropiados para funcionar de manera correcta.

Los requerimientos son los siguientes:

Componentes de Servidor

1.Servidor SQL Server con las siguientes características:

▪Servidor físico o virtual con un mínimo de 1 un procesador  y 2 core

▪8 GB de RAM o más

▪SQL Server versión 2014 o posterior, edición express,  estándard o enterprise

▪Espacio para base de datos y log mayor a 15 GB

2.Si elije la edición web, necesita además instalar un servidor Internet Information Services (IIS) con las siguientes características:

▪Servidor físico o virtual con un mínimo de 1 un procesador  y 2 core

▪8GB de RAM o más

▪IIS 8.5 o posterior

▪NET Framework 4.8

▪2 GB de espacio para la aplicación

En el caso de la edición Web, el servidor IIS puede ser el mismo donde se instaló la base de datos

 

Componente Cliente

Para la edición Windows, necesitará que los equipos donde usará la aplicación tengan las siguientes características:

▪Cualquiera de los siguientes sistemas operativos: Windows 7, Windows 8.x, Windows 10 o posterior, Windows Server 2008 R2  o posterior.

▪Mínimo 4GB de memoria RAM. Dependiendo de que aplicaciones instaladas puede necesistar más. Se recomiendan 8 GB.

▪NET Framework 4.8 instalado. Si no se encuentra, el instalador del sistema intentará instalarlo en línea, pero necesita una conexión a Internet.

▪Procesador dual core o superior. Intel I3 o equivalente de tercera generación o posterior es recomendado.

 

Para la edición Web, el equipo de los clientes debe ejecutar cualquiera de los siguientes navegadores

▪Google Chrome Version 66.0.XXX o posterior

▪Mozilla FireFox versión xxxx o posterior

▪Microsoft Edge versión Microsoft Edge 42.XX o posterior

▪Puede ejecutar la aplicación en PC de escritorio, portáttiles, chromebook, tablets y smartphone. Tome en cuenta que en el caso de los smartphone, aún cuando la aplicación es responsive, para las consultas multidimencionales complejas, puede ser complicada la visualización de la información.
